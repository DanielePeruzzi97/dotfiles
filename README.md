# dotfiles

## Requirements

- zsh + oh-my-zsh
- Install zsh-syntax-highlighting and zsh-history-substring-search
- Alacritty
- Tmux
- stow
- A nerd font
- xclip

## Utiliy script

Copy dotfiles.sh to /usr/local/bin/dotfiles

Usage:

```bash
dotfiles sync options
```
